# Razi Perfumes - Luxury Arabian Fragrances

A premium, fully responsive luxury e-commerce store clone built with **React 19**, **Vite 7**, and **CSS Modules**.

---

## ✨ Features
* **Brand Rebranding**: Rebranded to **Razi Perfumes** / **رازي** (with elegant typography and Arabic translations).
* **Luxury Aesthetic**: A dark, gold, and cream color palette with premium serif headings (`Cinzel`, `Cormorant Garamond`) and modern typography (`Montserrat`).
* **Micro-Animations**: Clean fade-ins, hover zooming on product cards, and keyframe-animated loaders.
* **Country Modal & Persistent Settings**: Remembers selected region/currency (KD, SAR, AED, etc.) and wishlist choices in `localStorage`.
* **Dynamic Cart System**: Sidebar basket drawer support for item management (adding/removing and quantity updates) with real-time total price calculation.
* **Mobile Responsive**: Fully optimized layout for smaller devices (collapsible footer accordions, responsive grid systems, and spacing optimization in the header).

---

## 🚀 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```
